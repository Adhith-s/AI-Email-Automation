# AI Email Automation System 🚀

## 📌 Overview
This project automates email generation using AI and sends emails to multiple recipients.

## 🔧 Features
- AI-based email content generation (HuggingFace)
- Bulk email processing using CSV
- Modular Python structure
- Simulation mode for safe testing

## 🛠️ Tech Stack
- Python
- Transformers (HuggingFace)
- Pandas

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python main.py

## 📁 Project Structure
AI-EMAIL-AUTOMATION/
│── data/
│── scripts/
│── main.py
│── requirements.txt

## 📬 Output
Generates professional emails and simulates sending to multiple users.
